Typography

Typography!

Using Sass

Variables

NameTypeDefault ValueDescription
$base-line-height Number 1.5

The base line height for your typeface.
$cap-height Number 0.68

The cap height for your typeface. Cap height is defined as the height of a capital letter above the baseline grid.
$ms-intervals Map ( default: 1.2, medium: 1.333, large: 1.4, )

A map of intervals to use to scale type at various breakpoints. For each item, the key should be the name of a breakpoint used in the $breakpoints map, and the value is an integer representing the interval to use. For example, 1.618 is the golden ratio.

Mixins

baseline

@mixin baselin) { }

Define a baseline grid for an element, based on the font size, line height, and cap height of the type.

font-size

@mixin font-size($n, $intervals) { }

Returns multiple font-size declarations at different breakpoints, using the breakpoints and scales defined in $ms-intervals.

NameTypeDefault ValueDescription
$n Number

Position on the scale to use. 1 is the starting point (1em).
$intervals Map $ms-intervals

A set of intervals to use.

Functions

modular-scale

@mixin modular-scale($n, $ratio) { }

Calculates a number at position $n on a modular scale, using $ratio to move up the scale.

NameTypeDefault ValueDescription
$n Number 1

Position on the scale to calculate. 1 is the starting point.
$ratio Number

Interval to use in calculation.
