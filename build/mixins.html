Mixins

Using Sass

Mixins

css-triangle

@mixin css-triangle($triangle-size, $triangle-color, $triangle-direction) { }

Creates a CSS triangle, which can be used for dropdown arrows, popup tails, and more. Use this mixin inside a &::before or &::after selector, to attach the triangle to an existing element.

NameTypeDefault ValueDescription
$triangle-size number

Width of the triangle.
$triangle-color color

Color of the triangle.
$triangle-direction keyword

Direction the triangle points. Can be top, right, bottom, or left.

hamburger

@mixin hamburger($width, $left, $top, $thickness, $gap, $color, $hover-color) { }

Creates a three-line menu icon, affectionately referred to as the "hamburger icon".

NameTypeDefault ValueDescription
$width number

Width of the icon, in rem units.
$left number|boolean

Left offset of the icon. Set to false to center the icon horizontally.
$top number|boolean

Top offset of the icon. Set to false to center the icon vertically.
$thickness number

Height of each line in the icon.
$gap number

Amount of space between each line.
$color color

Color of the lines.
$hover-color color

Color of the lines on hover.

clearfix

@mixin clearfi) { }

Uses the micro clearfix hack popularized by Nicolas Gallagher. Include this mixin on a container if its children are all floated, to give the container a proper height.

element-invisible

@mixin element-invisibl) { }

Makes an element visually hidden, but still accessible to keyboards and assistive devices.

element-invisible-off

@mixin element-invisible-of) { }

Reverses the CSS output created by the element-invisible() mixin.
